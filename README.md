# OperationSEA
**2024.6 컴퓨터 그래픽스 기말 프로젝트 (WEBGL + three.js)**

https://0-1-dec.github.io/OperationSEA/

WebGL과 Three.js를 이용하여 만든 슈팅 게임입니다.
점차 빠르게 소환되는 적 개체의 공격을 피해 제거하여 점수를 얻습니다.

유니티의 제작 방식에서 영감을 얻어 프리팹과 같은 요소를 본따서 제작하였습니다.

※우수작품으로 선정되어 외부 협력 기업에서 발표했던 프로젝트입니다.
